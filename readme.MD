# Lokify
A fun package to animate text in a Lokish way

## Usage
To use Lokify you simply need to import the component as well as the styles:

```JS
import React from 'react;

import  { Loki } from 'lokify';

const Thing = () => (
    <div>
        <Loki />
    </div>
)

```
